# Página de Registro

## Descripción

Formulario para crear nuevos usuarios en el sistema.

## Tipos de Usuario

- **Ganadero**: Propietario de animales
- **Institución**: Entidad institucional (SENASA, mappings, etc.)
- **Gremio**: Asociación de productores
- **Administrador**: Personal técnico del sistema

## Validaciones

- Email válido y único
- Password con requisitos de seguridad
- Cédula única en el sistema
- Selección de tipo de usuario
- Datos específicos según tipo
