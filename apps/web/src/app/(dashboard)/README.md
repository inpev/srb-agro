# Área de Dashboard

## Descripción

Área privada del sistema que requiere autenticación y permisos válidos.

## Características

- Requiere sesión activa
- Control de acceso por roles
- Navegación lateral (sidebar)
- Dashboard principal según tipo de usuario

## Roles y Permisos

- **Nacional**: Acceso total al sistema
- **Regional**: Acceso por gobernación
- **Municipal**: Acceso por municipio
- **Institución**: Acceso según institución
- **Gremio**: Acceso a datos de asociados
- **Ganadero**: Datos propios
- **Control**: Acceso a módulo de verificación
